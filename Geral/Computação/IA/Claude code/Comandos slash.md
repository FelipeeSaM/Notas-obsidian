## Comandos que todo usuário precisa conhecer

# Abrir os chats (como no copilot do vscode): /resume

| Comando       | O que faz                                                                                              | Como usar                                                                                                                                                                                                                                        |
| ------------- | ------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `claude`      | Inicia uma sessão interativa do Claude Code no terminal. É o principal ponto de entrada da ferramenta. | **Iniciar sessão vazia:**  <br>`claude`  <br>  <br>**Iniciar com prompt:**  <br>`claude "explique a arquitetura deste projeto"`  <br>  <br>**Continuar última conversa:**  <br>`claude -c`                                                       |
| `/compact`    | Resume e compacta a conversa atual para economizar contexto.                                           | **Compactar conversa:**  <br>`/compact`  <br>  <br>**Focar em um tema específico:**  <br>`/compact foque nos erros da API REST`  <br>  <br>**Preservar apenas decisões importantes:**  <br>`/compact mantenha apenas as decisões de arquitetura` |
| `/clear`      | Limpa completamente o histórico da conversa atual, liberando contexto.                                 | Use para recomeçar a conversa do zero sem sair da sessão.  <br>  <br>Exemplo:  <br>`/clear`                                                                                                                                                      |
| `/help`       | Lista todos os comandos disponíveis na sessão atual, incluindo comandos nativos, customizados e MCP.   | Use quando esquecer comandos ou quiser descobrir novos recursos.  <br>  <br>Exemplo:  <br>`/help`                                                                                                                                                |
| `/model`      | Permite trocar o modelo de IA utilizado na sessão atual.                                               | **Abrir seletor de modelo:**  <br>`/model`  <br>  <br>**Definir modelo ao iniciar:**  <br>`claude --model opus`  <br>`claude --model claude-sonnet-4-5-20250929`                                                                                 |
| `/init`       | Cria um arquivo `CLAUDE.md` com contexto e instruções do projeto.                                      | Use ao iniciar o Claude Code em um projeto novo.  <br>  <br>Exemplo:  <br>`/init`                                                                                                                                                                |
| `/fast`       | Ativa o modo rápido da sessão, priorizando velocidade e menor custo.                                   | Use para tarefas simples e repetitivas como pequenas correções e formatações.  <br>  <br>Exemplo:  <br>`/fast`                                                                                                                                   |
| `/stats`      | Mostra métricas rápidas da sessão atual, como consumo e uso de contexto.                               | Use em tarefas grandes ou conversas longas para monitorar o estado da sessão.  <br>  <br>Exemplo:  <br>`/stats`                                                                                                                                  |
| `/usage`      | Exibe consumo de tokens e custo estimado da sessão.                                                    | Use para acompanhar custos e consumo antes de continuar tarefas pesadas.  <br>  <br>Exemplo:  <br>`/usage`                                                                                                                                       |
| `/doctor`     | Executa diagnóstico completo da instalação do Claude Code.                                             | Use para investigar problemas de conexão, autenticação, MCP ou dependências.  <br>  <br>Exemplo:  <br>`/doctor`                                                                                                                                  |
| `Ctrl + C`    | Interrompe input ou geração em andamento.                                                              | Pressione `Ctrl + C` no terminal.                                                                                                                                                                                                                |
| `Esc`         | Cancela a geração atual da IA.                                                                         | Pressione `Esc` durante a geração da resposta.                                                                                                                                                                                                   |
| `Ctrl + L`    | Limpa a tela do terminal.                                                                              | Pressione `Ctrl + L`.                                                                                                                                                                                                                            |
| `Tab`         | Liga ou desliga o modo _extended thinking_.                                                            | Pressione `Tab` durante a sessão.                                                                                                                                                                                                                |
| `Shift + Tab` | Alterna os modos de permissão.                                                                         | Pressione `Shift + Tab`.                                                                                                                                                                                                                         |
