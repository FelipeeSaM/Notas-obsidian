## Planejamento:
 -  O que fazer

| O que é       | modo de operacao do Claude Code            |
| ------------- | ------------------------------------------ |
| Onde acontece | interface (vocé vé e controla)             |
| Proposito     | definir o que fazer antes de editar        |
| Controle      | Shift+Tab ou /plan                         |
| Resultado     | plano em Markdown para aprovar ou rejeitar |
| Analogia      | arquiteto desenhando a planta              |
Aqui podemos combinar o modo de planning com alguns comandos de pesquisa:

| Read                 | ver arquivos               |
| -------------------- | -------------------------- |
| LS                   | listar diretorios          |
| Glob e Grep          | buscar arquivos e conteudo |
| Task                 | subagentes de pesquisa     |
| WebSearch e WebFetch | pesquisa web               |
| AskUserQuestion      | perguntar para vocé        |
