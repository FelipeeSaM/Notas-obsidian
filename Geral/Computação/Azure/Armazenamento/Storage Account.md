# Storage Account

Storage Account é o serviço base de armazenamento do [[Azure]].

## 📌 O que é
Uma Storage Account é um contêiner que agrupa diferentes tipos de armazenamento na nuvem, como arquivos, blobs e filas.

Ela funciona como um “ponto central” onde você cria e gerencia todos os recursos de armazenamento.

## 📌 Para que serve
- Armazenar arquivos
- Armazenar dados não estruturados
- Servir conteúdo estático
- Integrar com aplicações e serviços

## 📌 Tipos de armazenamento dentro dela
- Blob Storage → arquivos e dados não estruturados
- File Storage → compartilhamento de arquivos
- Queue Storage → filas de mensagens
- Table Storage → dados NoSQL simples

## 📌 Características
- Alta durabilidade e disponibilidade
- Escalabilidade automática
- Acesso via HTTP/HTTPS
- Integração com outros serviços do Azure

## 📌 Exemplo
Uma aplicação pode usar:
- Blob Storage para imagens
- Queue para processamento assíncrono

Tudo dentro de uma única Storage Account
