# Contêineres

Forma de empacotar aplicações para execução consistente no [[Azure]].

## 📌 O que é
Uma unidade leve que contém aplicação + dependências.

## 📌 Para que serve
- Garantir que a aplicação rode igual em qualquer ambiente
- Facilitar deploy
- Isolar aplicações

## 📌 Características
- Baseado em [[Docker]]
- Portável
- Rápido de iniciar

## 📌 Serviços relacionados no Azure
- Azure Container Instances (ACI)
- Azure Kubernetes Service (AKS)

## 📌 Quando usar
- Microserviços
- Sistemas escaláveis
- Ambientes padronizados
