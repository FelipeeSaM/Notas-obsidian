# Virtual Machines

Máquinas virtuais do [[Azure]] que simulam servidores físicos.

## 📌 O que é
Um serviço IaaS que permite criar servidores completos na nuvem.

## 📌 Para que serve
- Rodar aplicações com controle total do sistema
- Instalar qualquer software
- Simular ambiente on-premise

## 📌 Características
- Controle total do SO (Linux/Windows)
- Configuração manual
- Maior responsabilidade de manutenção

## 📌 Quando usar
- Sistemas legados
- Aplicações que exigem controle do servidor
- Cenários específicos de infraestrutura
