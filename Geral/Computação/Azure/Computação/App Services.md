# App Services

Serviço do [[Azure]] para hospedar aplicações web sem gerenciar infraestrutura.

## 📌 O que é
Uma plataforma PaaS para rodar aplicações web, APIs e backends.

## 📌 Para que serve
- Hospedar APIs .NET
- Publicar sites e aplicações web
- Fazer deploy rápido sem configurar servidor

## 📌 Características
- Escala automática
- Deploy simples (Git, CI/CD)
- Suporte a várias linguagens (.NET, Node, Java)

## 📌 Quando usar
- Aplicações web tradicionais
- APIs REST
- Sistemas que não precisam de controle de infraestrutura
