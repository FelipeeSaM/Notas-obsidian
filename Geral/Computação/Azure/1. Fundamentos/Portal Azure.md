# Portal Azure

O Portal Azure é a interface web do [[Azure]].

## 📌 O que é
Uma aplicação web usada para gerenciar recursos visualmente.

## 📌 Para que serve
- Criar recursos
- Configurar serviços
- Monitorar aplicações
- Gerenciar custos

## 📌 Conceito chave
É a forma mais simples de interagir com o Azure, sem usar código.

## 📌 Alternativas
Além do portal, você também pode usar:
- CLI (linha de comando)
- Scripts (automação)

## 📌 Quando usar
- Exploração inicial
- Testes
- Configurações rápidas
