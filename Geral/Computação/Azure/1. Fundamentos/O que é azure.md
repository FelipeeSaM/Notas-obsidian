# O que é Azure

O [[Azure]] é uma plataforma de computação em nuvem da Microsoft.

## 📌 O que é
Um conjunto de serviços que permite criar, hospedar e gerenciar aplicações e infraestrutura na nuvem.

## 📌 Para que serve
- Hospedar aplicações (APIs, sites, sistemas)
- Armazenar dados
- Executar processos e automações
- Escalar sistemas sob demanda

## 📌 Conceito chave
Ao invés de comprar servidores físicos, você utiliza recursos sob demanda pela internet.

## 📌 Modelos de serviço
- IaaS (Infraestrutura como serviço)
- PaaS (Plataforma como serviço)
- SaaS (Software como serviço)
