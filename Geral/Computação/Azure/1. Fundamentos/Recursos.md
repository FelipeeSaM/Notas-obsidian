# Recursos

Recursos são os serviços individuais que você cria dentro do [[Azure]].

## 📌 O que é
Qualquer componente criado na plataforma, como:
- Máquina virtual
- Banco de dados
- API
- Storage

## 📌 Para que serve
São os blocos que compõem sua aplicação na nuvem.

## 📌 Conceito chave
Tudo no Azure é um recurso.

## 📌 Características
- Sempre pertencem a um [[Resource Groups]]
- São criados em uma [[Regiões]]
- Podem ser gerenciados individualmente

## 📌 Exemplos
- App Service
- Azure SQL Database
- Blob Storage
