# Azure SQL Database

Serviço de banco de dados relacional gerenciado do [[Azure]].

## 📌 O que é
Uma versão na nuvem do SQL Server, totalmente gerenciada pelo Azure.

Você não precisa instalar, configurar ou manter o banco — o Azure cuida disso.

## 📌 Para que serve
- Armazenar dados estruturados
- Aplicações com modelo relacional
- APIs que utilizam SQL

## 📌 Características
- Alta disponibilidade
- Backup automático
- Escalabilidade
- Segurança integrada

## 📌 Vantagem principal
Você foca apenas nos dados e queries, não na infraestrutura.

## 📌 Exemplo
Uma API .NET pode:
- salvar usuários
- consultar dados
- executar queries SQL

Tudo usando Azure SQL Database
