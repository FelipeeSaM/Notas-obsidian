# Cosmos DB

Banco de dados NoSQL distribuído do [[Azure]].

## 📌 O que é
Um banco de dados que armazena dados não estruturados ou semi-estruturados, geralmente no formato JSON.

Diferente de bancos relacionais, não usa tabelas fixas.

## 📌 Para que serve
- Aplicações escaláveis globalmente
- Sistemas com grande volume de dados
- Aplicações com estrutura flexível

## 📌 Características
- Distribuição global
- Baixa latência
- Alta escalabilidade
- Suporte a múltiplos modelos (documento, chave-valor, grafo)

## 📌 Exemplo
Uma aplicação pode armazenar:

```json
{
  "id": "1",
  "nome": "Felipe",
  "habilidades": ["C#", ".NET", "Azure"]
}
```

## 📌 Quando usar

- Dados dinâmicos
- Alta escala
- Aplicações modernas

## 📌 Quando NÃO usar

- Sistemas fortemente relacionais
- Queries complexas com JOINs
