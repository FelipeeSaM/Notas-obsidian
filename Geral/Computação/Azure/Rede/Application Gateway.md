# Application Gateway

Application Gateway é um balanceador de carga avançado do [[Azure]] para aplicações web.

## 📌 O que é
Um serviço que atua como intermediário entre o usuário e a aplicação, oferecendo roteamento inteligente e segurança.

## 📌 Para que serve
- Roteamento baseado em URL
- Proteção de aplicações web
- Balanceamento de carga em nível HTTP/HTTPS

## 📌 Características
- Suporte a SSL (HTTPS)
- Web Application Firewall (WAF)
- Roteamento baseado em regras

## 📌 Exemplo
Uma aplicação pode ter:

- /api → backend
- /site → frontend

👉 O Application Gateway direciona cada rota corretamente
